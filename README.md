# Лабораторна робота №1. Фронтенд
## Варіант №1
**Тема:** Менеджер заявок на доступ до лабораторії
**Ключові сутності:** Користувачі (Users), Заявки на доступ (AccessRequests)
## Функціонал
*   **Додавання заявок:** Форма з полями "Ім'я користувача", "Дата/Час", "Тип доступу", "Коментарі".
*   **Відображення:** Динамічне відображення даних у таблиці (елементи `<thead>` та `<tbody>`).
*   **Валідація:** Мінімальна перевірка обов'язкових полів та дати.
*   **Додатково (на "добре" та "відмінно"):** Видалення записів, підсвічування помилок, збереження в `localStorage`.
## Запуск
Проєкт розроблено як статичний фронтенд і може бути запущений локально.
**Спосіб 1: Через розширення VS Code (рекомендовано):**
1.  Встановіть розширення "Live Server" у VS Code.
2.  Натисніть правою кнопкою миші на файл `index.html` і виберіть "Open with Live Server".
**Спосіб 2: За допомогою Python сервера:**
Перейдіть у папку проєкту в терміналі та виконайте команду:
```bash
python -m http.server 8080
# Або спробуйте так, якщо попередня не спрацює:
python3 -m http.server 8080
